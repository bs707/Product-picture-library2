Get Started in 5 Minutes

Follow this quick start guide to integrate SocialCam Master Pro into your workflow.

Installation

Open Blender → Edit > Preferences > Add-ons → Install → Select .zip file → Enable "SocialCam Master Pro". Access via 3D Viewport Sidebar (Press N).

Quick Start Workflow

1. Set Aspect Ratio: Select a resolution preset (e.g., 9:16 for TikTok)
2. Compose Your Shot: Enable Composition Guides (Rule of Thirds)
3. Save Your View: Adjust camera and click Record View
4. Apply FX: Toggle CRT Mode or adjust Saturation in viewport

Expert Tips & Shortcuts

• Press N: Toggle Sidebar for Camera Tool panel
• Auto-Sync: FX settings automatically update Compositor nodes
• Lock View: Use "Lock Camera View" for intuitive navigation
• Bookmark Sync: Timeline markers automatically sync with camera bookmarks
